# Modern Calculator Web App

A fully functional calculator web application with a clean, dark mode UI and responsive design.

## Features

- **Modern Dark Mode UI**: Sleek design with rounded corners and shadows
- **Responsive Layout**: Works on both mobile and desktop devices
- **Keyboard Support**: Use your keyboard for input (numbers, operators, Enter for =, Escape/C for clear)
- **Error Handling**: Displays "Error" for invalid operations (like division by zero)
- **Clean Button Grid**: Intuitive 4x4 button layout with special buttons spanning two columns

## How to Use

1. **Open the App**:
   - Simply open the `index.html` file in any modern web browser
   - Or serve the files using a local web server (recommended for best experience)

2. **Basic Operations**:
   - Click the number buttons (0-9) to input numbers
   - Use the operator buttons (+, -, *, /) to perform calculations
   - Press the equals (=) button to see the result

3. **Clear the Calculator**:
   - Click the 'C' button to clear the current input
   - Press Escape or C on your keyboard

4. **Keyboard Shortcuts**:
   - Type numbers directly on your keyboard
   - Use +, -, *, / for operators
   - Press Enter or = for calculation
   - Press Escape or C to clear

## Technical Details

- **HTML5**: Semantic structure with proper button grid
- **CSS3**: Modern styling with CSS Grid, transitions, and responsive design
- **Vanilla JavaScript**: No dependencies, pure JavaScript implementation
- **Cross-browser compatible**: Works on Chrome, Firefox, Safari, and Edge
